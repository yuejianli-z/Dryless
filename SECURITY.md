# Security Policy

## Privacy and Data Handling

Dryless is intended to run locally. The app processes camera frames in memory and does not upload or store camera images or video.

Local settings and numeric blink statistics are stored under:

```text
~/.blink_reminder/
```

## Reporting Security Issues

If you find a security or privacy issue, please do not disclose it publicly first. Open a private security advisory on GitHub if available, or contact the maintainer through the repository's preferred contact channel.

## Known Distribution Notes

Release executables may be unsigned. Windows SmartScreen or antivirus software may display warnings for unsigned executables, especially for new open-source projects.
