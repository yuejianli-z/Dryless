# Dryless v0.1.0

Initial public release of Dryless.

## Download

Download `Dryless.exe` from the assets below and double-click to run it on Windows.

## Notes

- No Python installation is required for the release executable.
- A webcam is required.
- The app processes camera frames locally in memory.
- It does not upload or store camera images or video.
- The executable is currently unsigned, so Windows SmartScreen or antivirus software may show a warning.
